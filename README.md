# businesscentral-eventing

This repository contains Business Central AL extensions related to event-driven integration.

## Extensions

| Extension | Description |
| --- | --- |
| [bc-eventing](apps/business-central/bc-eventing/README.md) | Captures record-level events (insert, modify, delete, rename) and publishes them to an external event hub using the outbox pattern. |
